Release Notes

MonsterCalc iPhone 1.0.0.

Version 1.0.0

  • Built a side-by-side scratchpad and live results layout for iPhone.
  • Added custom Text, Calc, Math, Convert, EE, and Prog keyboards.
  • Added ghost hints, line-number gutter support, and tap-to-insert lineX references from live results.
  • Added math, programming, and EE helper coverage including bitset and findrdiv.
  • Added local HTML documentation and an automated XCTest suite for the core engine and view model.
