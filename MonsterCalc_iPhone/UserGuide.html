MonsterCalc iPhone

User guide for the MonsterCalc scratchpad calculator.

How It Works

Type one expression per line in the scratchpad. The live results pane evaluates each line in order. Earlier successful results become available to later lines.

Assignments

Assign a value to a variable and reuse it later.

x = 2*pi
y = x + 1

Previous Result

Use ans to refer to the last successful result.

5
ans * 10

Line References

Use line2, line7, and so on to reference earlier result lines directly.

x = 300
vdiv(5, 10k, 10k)
line2 + x

Scratchpad Basics

  • Blank lines are allowed.
  • Start a comment with #.
  • Sheets are autosaved as you work. Nonblank sheets are kept in a rolling list of the 10 most recently edited sheets.
  • Tap a live result to insert its matching lineX token at the cursor.
  • The editor shows ghost hints for built-in functions while you type.
  • Incomplete expressions such as abs( are treated as in-progress input, not hard errors.
Tip: if a helper auto-inserts a closing ), keep typing. The caret stays inside the parentheses so you can finish the argument naturally.

Sheets

Autosave

There is no Save button. MonsterCalc automatically saves the current nonblank sheet and updates its last-edited time as you work.

New Sheet

Tap the + button beside the hamburger menu to start a fresh sheet. The current nonblank sheet is autosaved before the new sheet opens.

Load Sheet

Open the hamburger menu and choose Load Sheet to see recent sheets. Each entry shows the last edited date and a preview of the first nonblank line.

Keyboard Modes

Mode Purpose Examples
Text General typing for variables, function names, comments, and mixed expressions. x = 2*pi, # note, findrdiv

Math π, √, logs, trig, probability, and engineering suffixes. sqrt(2), cdf(1.96), 10u
Convert Choose a unit family, insert units quickly, and build conversions like 70 F to C. mm, in, 70 F to C
EE Electrical helper functions. vdiv(5, 10k, 10k), findrdiv(5, 3.3)
Prog Programming helpers, bases, and bit operations. 0xff xor 0x0f, bitpunch(1, 7, 1)
